# MeetHive (Frontend)
MeetHive is a niche event marketplace(Starting with games night events) where organizers can create branded profiles, host events, sell tickets(single, group) and attendees can join, chat and share event experiences.

## Tech Stack
-React
-Vite
-TypeScript
-TailWind CSS

## Installation and Setup

- Create the folder locally and publish to github
- Create vite and choose react and typeScript as the frameWork and language technologies
- install dependencies
      **npm install**
- Start development server 
      **npm run dev**

# Planned Features
- User authentication and organizer profiles
- Event creation and listing
- Ticket purchases and QR-based scanning
- Real-time chat rooms for events
- Photo uploads with moderation
- AI-powered event chatbots
